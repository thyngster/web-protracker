# ProTracker Clone

A web-based MOD player that replicates the classic Amiga ProTracker 2.3 interface.

## Features

- Authentic ProTracker 2.3 UI with Amiga-style 3D buttons
- 4-channel MOD file playback using Web Audio API
- Real-time waveform visualization (Quadrascope)
- Pattern editor display
- Sample list view

## Tech Stack

- Vue 3 with TypeScript
- Vite
- Tailwind CSS v4
- Web Audio API with AudioWorklet

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Usage

1. Click "DISK OP." to load a .MOD file
2. Click "PLAY" to start playback
3. Click "STOP" to stop playback
